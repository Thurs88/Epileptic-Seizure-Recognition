# Epileptic-Seizure-Recognition

Диагностика эпилепсии по ЭЭГ-сигналам.  
Оригинальный набор данных: https://archive.ics.uci.edu/ml/datasets/Epileptic+Seizure+Recognition.  
Перед запуском проекта необходимо выполнить скрипт 'create_project_env.py', который создает необходимые папки под проект.  
Для получения представления о данных см. тетрадку 'EDA+Baseline.ipynb' (папка notebooks).  
Итоговая логика предобработки данных и обучения модели - в папке code.  
Для запуска пайплайна классификации выполнить скрипт 'main.py'.
